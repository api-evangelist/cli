# Command Line Interface (cli)

A curated index of services, tooling, and open source solutions for API command line interfaces, HTTP clients, and terminal-based API tools. This collection covers HTTP clients, API-specific CLIs, spec and linting CLIs, cloud provider CLIs, platform CLIs, and infrastructure management tools.

**APIs.json:** [apis.yml](https://raw.githubusercontent.com/api-evangelist/cli/refs/heads/main/apis.yml)

## Scope

- **Type:** Index
- **x-type:** topic

## Tags

CLI, Command Line Interface, Developer Tools, HTTP Client, Terminal

## Network

This is a topic index aggregating CLI-related repos across the API Evangelist network. The full list of network entries lives in `apis.yml` under `network:` and includes HTTP clients (curl, HTTPie, xh, Hurl), API design CLIs (Buf, Kiota, OpenAPI Generator, Redocly, Spectral, Vacuum), API client CLIs (Bruno, Insomnia, Postman/Newman, Hoppscotch), cloud CLIs (AWS, Azure, Google Cloud, Cloudflare, Fly.io, Netlify, Vercel), platform CLIs (Docker, Helm, kubectl, Pulumi, Terraform, Supabase), and provider CLIs (GitHub CLI, Stripe, Twilio).
